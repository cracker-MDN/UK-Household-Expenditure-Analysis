# Data

## Dataset

**Name:** Living Costs and Food Survey (LCF) 2013 - Teaching Dataset  
**Source:** Office for National Statistics (ONS)  
**Sample:** 5,144 UK households

## ⚠️ Not Included

The dataset is **not included** in this repository due to licensing restrictions.

## How to Access

The LCF teaching dataset is available from the UK Data Service:

1. Visit [UK Data Service](https://ukdataservice.ac.uk/)
2. Register for an account (free for academic users)
3. Search for "Living Costs and Food Survey 2013"
4. Download the teaching dataset version

**Direct Link:** [SN 7472](https://beta.ukdataservice.ac.uk/datacatalogue/studies/study?id=7472)

## Variables Used in This Analysis

| Variable | Description | Type |
|----------|-------------|------|
| `Expenditure` | Total weekly household spending (£) | Continuous |
| `NS - SEC 8 Class of household reference person` | Occupational classification | Nominal (5 categories) |
| `Tenure type` | Housing arrangement | Nominal (3 categories) |
| `Number of adults` | Adults in household | Ordinal (4 categories) |
| `Number of children` | Children in household | Ordinal (3 categories) |

## Data Summary

| Statistic | Value |
|-----------|-------|
| Total households | 5,144 |
| Expenditure range | £31 - £1,175/week |
| Mean expenditure | £480/week |
| Median expenditure | £420/week |
